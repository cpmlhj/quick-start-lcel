# LCEL-RAG 项目

## 技术栈

- LangChain
- OpenAI
- ChromaDB
- Playwright
- TypeScript

## 功能

1. RAG应用：基于向量检索的智能问答系统
2. 多核心代码生成：支持Java、Python、TypeScript多语言代码生成

## 运行方式

```bash
# 安装依赖
npm install

# 运行RAG应用
npm run start

# 运行多核心代码生成
npm run code
```

## 环境要求

- Node.js
- TypeScript
- OPENAI_API_KEY（需配置环境变量）
- HTTP_PROXY (可选代理地址)
